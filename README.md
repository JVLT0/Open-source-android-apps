# Por que?
> Esta é uma coleção de aplicativos Android que são de código aberto.
> Acredito que podemos encontrar algo interessante e adquirir habilidades úteis com um aplicativo de código aberto.
> Esta ideia vem de [este projeto](https://github.com/dkhamsing/open-source-ios-apps)


# Como?
> Vou enviar projetos incríveis quando encontrar algum.
> É BEM-VINDO para você [star](https://github.com/pcqpcq/open-source-android-apps/stargazers) e [fork](https://github.com/pcqpcq/open-source-android-apps#fork-destination-box) este projeto.
> e você souber de algo que não está aqui, seria ótimo se você pudesse fazer um [Pull Request](https://github.com/pcqpcq/open-source-android-apps/pulls).  
> Todos os aplicativos são classificados em várias categorias.
